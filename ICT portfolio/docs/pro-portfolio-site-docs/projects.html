Password Manager (Python)

Simple local password manager to practice Python fundamentals and secure storage. Uses hashing (e.g., SHA‑256) and hidden input with getpass. Future: switch to libsodium/cryptography for encryption and add a small GUI.

Phishing Email Analysis

Beginner workflow to analyze suspicious emails: full headers, SPF/DKIM, WHOIS lookup, and link checks in VirusTotal from a safe machine. Document red flags and create a checklist.

Vulnerability Scan (Safe Lab)

Nmap & Nessus Essentials against a local vulnerable VM (e.g., Metasploitable2). Identify open ports/services, map CVEs, and export a short findings report.

Secure Network Lab

Home lab using VirtualBox + pfSense + two VMs. Create firewall rules, test blocking, and capture packets with Wireshark to observe allowed/blocked flows.

Basic Log Monitoring (Splunk)

Ingest Windows Security logs into Splunk Free and build a dashboard for failed logins + basic alerts. Learn SIEM pipelines and indexes.
